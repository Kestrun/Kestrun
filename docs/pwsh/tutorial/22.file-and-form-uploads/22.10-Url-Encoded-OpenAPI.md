2. Server: Create the host and bind a listener.
3. OpenAPI: Add document info and contact metadata.
4. Storage: Set the default upload directory.
5. Model: Define `UrlEncodedForm` with required and multi-value fields.

## Try it

```powershell
$body = 'name=Kestrun&role=admin&role=maintainer'
Invoke-RestMethod -Method Post -Uri 'http://127.0.0.1:5000/form' -ContentType 'application/x-www-form-urlencoded' -Body $body
```

```bash
curl -H "Content-Type: application/x-www-form-urlencoded" \
  -d "name=Kestrun&role=admin&role=maintainer" \
  http://127.0.0.1:5000/form
```

## Troubleshooting

- 415 / Unsupported Content-Type: Ensure `application/x-www-form-urlencoded` is sent.
- Missing fields: Verify the body uses `name=` and `role=` keys.
