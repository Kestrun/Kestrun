## Step-by-step

1. Logger: Register a console logger for visibility.

## Try it

```powershell
$client = [System.Net.Http.HttpClient]::new()
$content = [System.Net.Http.MultipartFormDataContent]::new()
$content.Add([System.Net.Http.StringContent]::new('Hello from client'), 'note')
$bytes = [System.Text.Encoding]::UTF8.GetBytes('sample file')
$fileContent = [System.Net.Http.ByteArrayContent]::new($bytes)
$fileContent.Headers.ContentType = [System.Net.Http.Headers.MediaTypeHeaderValue]::Parse('text/plain')
$content.Add($fileContent, 'file', 'hello.txt')
$resp = $client.PostAsync('http://127.0.0.1:5000/upload', $content).Result
$resp.Content.ReadAsStringAsync().Result
```

```bash
curl -F "note=Hello from client" -F "file=@hello.txt;type=text/plain" http://127.0.0.1:5000/upload
```

## Troubleshooting

- 415 / Unsupported Content-Type: Ensure the request is `multipart/form-data`.
- Missing fields: Confirm the part names are `note` and `file`.
